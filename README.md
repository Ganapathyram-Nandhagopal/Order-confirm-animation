# 🎯 Order Confirm Animation
A modern animated order confirmation button built with HTML, CSS/SCSS, JavaScript, and jQuery. Features a stylish truck animation sequence, complete with smooth transitions, responsive design, and a premium interactive experience.

## ✨ Features
- 🚀 Pure Frontend Component
- 🎭 Smooth Truck and Order Processing Animations
- ⚡ Interactive Button States
- 🎨 Modern UI Design
- 📱 Responsive Layout
- ♿ Accessibility-Friendly
- 🔧 Easy Customization
- 🌐 GitHub Pages Compatible

## 🛠️ Technologies Used
- HTML5
- CSS3
- SCSS
- JavaScript (ES6)
- jQuery
- Vite

## 🎬 Animations Included

| Animation | Description |
|-----------|-------------|
| Button Transition | Smooth transition on click to animation state |
| Truck Animation | Animated delivery truck arriving and leaving |
| Box Loading | Box animated into the truck |
| Line Effects | Speed lines indicating truck movement |
| Checkmark Animation | Success checkmark drawn smoothly |

## 🚀 Getting Started

### Clone the Repository
```bash
git clone https://github.com/Ganapathyram-Nandhagopal/Order-confirm-animation.git
```

### Navigate to the Project
```bash
cd Order-confirm-animation
```

### Install Dependencies
```bash
npm install
```

### Run the Project
```bash
npm run dev
```
Open `http://localhost:5173` in your preferred browser.

## 📱 Responsive Design
